# Task 3: Dashboard Development (Power BI)

## Objective:
To create an interactive dashboard using Power BI for employee data analysis.

## Dataset:
Employee dataset containing Department, Salary, Experience, City, and Performance Score.

## Visualizations Created:
- Clustered Column Chart (Department vs Avg Salary)
- Stacked Column Chart (City vs Employee Count by Department)
- Column Chart (Experience vs Salary)
- Donut Chart (Performance Distribution)
- KPI Cards (Total Employees, Avg Salary, Max Salary)
- Filters (City and Department)

## Tools Used:
- Microsoft Power BI

## Conclusion:
The dashboard provides clear and interactive insights into employee distribution, salary trends, and performance.
